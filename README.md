# 🚀 Niyantra AI Dashboard

A modern AI-powered dashboard web application built using React, Vite, and Tailwind CSS.  
Designed for real-time data monitoring, visualization, and interactive user experience.

---

## ✨ Features

- 📊 Interactive Dashboard UI
- ⚡ Fast performance with Vite
- 🎨 Modern glassmorphism design (custom UI styles)
- 🔄 Real-time data fetching integration
- 🧭 Multi-page routing (Landing + Dashboard)
- 🛡️ Error Boundary for stability
- ⚙️ Global state management using Context API

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Routing:** React Router
- **Styling:** Tailwind CSS + Custom CSS
- **Build Tool:** Vite
- **State Management:** Context API
